# NeuroBloom 

NeuroBloom is a fully functional ReactJS web application built for **Project Phase 2**. It provides an empathetic, highly intuitive platform designed to help users identify common mental health struggles and explore simple, low-pressure cognitive and physical solutions.

##  Features
- **Responsive Navbar:** Includes a custom animated mobile hamburger menu for flawless navigation on small screens.
- **Stormy Minds (Problems):** Presents 8 mental health challenges using simplified, conversational language to ensure maximum accessibility for users under stress.
- **Healing Pathways (Solutions):** Offers 8 actionable, comforting micro-habits and techniques to help users reset their emotional state.
- **Premium Local Visuals:** Clean architecture utilizing high-resolution assets imported locally from the `src/assets/` folder.

---

##  Tech Stack & Architecture
- **Frontend Framework:** ReactJS (Functional Components & Hooks)
- **Styling:** Custom Vanilla CSS3 (Custom Responsive Grids & Flexbox layouts)
- **Icons:** React Icons (`fa` and `io5` libraries)
